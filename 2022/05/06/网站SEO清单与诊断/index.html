网站SEO清单与诊断

发表于 2022-05-06 更新于 2022-05-09 分类于 网站优化

网站优化对于搜索引擎排名的重要性不言而喻，同时SEO又是一项系统性的工作，涉及的项目很多。如果去从单点优化，想到什么做什么，那么很容易忽略掉真正重要的事情，也会带来很多遗漏。

在开始网站优化之前，很有必要准备一份优化清单，逐项检查开展。


01 SEO诊断工具

Rank Math可以一键诊断网页seo的情况，并且给出相应的评分。当然，除了Rank Math外，还有不少的工具。

这个网站可以自动检测网站需要从哪些方面去优化，https://sitechecker.pro/

  • meta description

关于meta description：How to create the right meta description

简要总结：

（1）元描述是对整个页面的概述；

（2）Google可能会在搜索结果中显示元描述，从而起到增加点击率的作用。但是没有直接证据证明元描述对排名有积极作用，Google也不一定会展示meta description.

（3）怎样写好一个meta description？

  • 保持155个字符以内，120-156最佳；
  • 直接清晰真实地描述页面内容，站在用户角度想他们最想知道什么；
  • 插入核心关键词（如果用户搜索词相同，Google会加粗显示）；
  • 产品页的元描述可以增加类似于厂商，规格，型号，功效等描述；
  • 禁止使用相同的元描述，这样可能会被认为页面内容重复；
  • 如果页面太多，可以只写关键页面的meta description；

（5）社交分享时有可能展示meta description；
