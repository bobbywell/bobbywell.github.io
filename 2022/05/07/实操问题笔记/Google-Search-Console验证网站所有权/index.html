Google Search Console验证网站所有权

发表于 2022-05-07 更新于 2022-05-08 分类于 网站优化

在注册好Google Search Console账号后，我们需要将自己网站添加进来。

添加网站之前，需要完成对网站所有权进行验证。


具体的验证方式可参考以下文章：

什么是谷歌站长工具(Google Search Console)：SEO教程

(2022）Google站长工具教程- Google search console使用教程

（图文）Google search console验证网站所有权


需要指出的是，如果你的网站已经开启了CDN加速，验证需要添加的TXT记录是没有用的，验证无法通过。

那该怎么办呢？

方法很简单，就是到你现在DNS管理服务器下面去添加，而不是最初的域名服务商。

比如我用了Cloudfare加速，我只需要到我的cloudfare账号的DNS管理界面添加一条TXT记录就可以了。

添加成功后，Google search console秒过。
